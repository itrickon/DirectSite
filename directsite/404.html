404 Not Found

Запрошенная страница не найдена.

Если вы видите эту страницу вместо сайта Django, возможно:

  • Django ещё не запустился правильно
  • Проблемы с конфигурацией Passenger
  • Неправильные пути в настройках

Проверьте:

  1. Файл passenger_wsgi.py существует и содержит правильные пути
  2. Виртуальное окружение установлено и активировано
  3. Все зависимости установлены: pip install -r requirements.txt
  4. Перезапустите приложение через restart.txt
