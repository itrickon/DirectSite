500 Internal Server Error

Произошла ошибка при обработке запроса.

Возможные причины:

  • Не установлен Python или виртуальное окружение
  • Не установлены зависимости Django
  • Ошибка в настройках (settings.py)
  • Проблемы с базой данных

Что делать:

  1. Проверьте логи ошибок в панели Beget
  2. Убедитесь, что виртуальное окружение активировано
  3. Выполните команды на сервере: 
    cd /home/i/itrickon/itrickon.beget.tech/public_html
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py collectstatic --noinput
  4. Перезапустите приложение через restart.txt
